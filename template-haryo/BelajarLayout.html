Ya Hallo, ketemu lagi, sudah lama tidak post yah. kali ini saya akan berbagi cara mereset password linux mint bila terjadi lupa password. kadang kita sering kasih password yang ribet yang pada akhirnya kita lupa sendiri passwordnya. mau ganti password gimana, kalau password lama nya aja lupa. di tambah lagi tidak bisa membuka terminal seperti biasa. ok langsung saja ikuti langkah ane ya.

  1. bila sudah lupa passwordnya. langkah pertama dengan me reboot PC. tapi di usahain dulu inget inget passwordnya. kalo kepepet ya sudah ikutin aja
  2. ketika sudah berada pada menu GRUB atau bootloader. klik tombol "E".
  3. maka akan muncul suatu semacam program. arahkan scroll dengan tombol arah ke tanda merah, antara qulet splash dan $vt_handoff
  4. ketikkan "rw init=/bin/bash" tanpa tanda petik. ketikkan di antara qulet dan $vt_handoff, sehingga menjadi qulet splash rw init=/bin/bash $vt_handoff, jika sudah pencet F10 untuk mereboot
  5. ketika sudah pencet F10. maka pc akan mereboot lalu muncul terminal, yah seperti terminal. sekarang tinggal ketikkan "passwd user_name" tanpa petik. user_name yang digunakan adalah user name yang akan di ganti passwordnya.
  6. masukkan password baru. lalu masukkan kembali password untuk konfirmasi password. setelah itu restart dengan perintah sudo reboot, lalu enter
  7. ok password sudah selesai di reset. sudah dulu dari saya. samapi ketemu lain waktu, bay bay :v
